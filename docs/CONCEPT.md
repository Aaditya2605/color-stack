# Color Stack Concept

## Pillars
- Satisfying timing-based stacking.
- Color matching/combos for score bursts.
- Short sessions with escalating intensity.

## Mechanics (Draft)
- Perfect drop: bonus points and combo multiplier.
- Color rule: drop matches top color to keep stack stable; mismatches shrink the stack slightly.
- Hazard: speed increases every 10 drops.

## Progression
- Daily challenge seed.
- Cosmetic themes unlocked via score milestones.
